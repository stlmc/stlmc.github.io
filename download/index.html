Download the STLmc Tool

The STLmc tool is written in Python. The tool is available for Linux (64-bit) and macOS (64-bit). There are two ways to download the tool.

  1. Installation from Source Code
  2. Installation from Docker

Installation from Source Code

Prerequisites

To build the STLmc tool, the followings are required:

  • Python version 3.8.x or newer
  • JAVA 8: https://openjdk.java.net/install/
  • Python3-pip: https://pypi.org/

Installation

  1. Download Yices2 (https://github.com/SRI-CSL/yices2):

    MacOS

    To install on Darwin, use homebrew:

     brew install SRI-CSL/sri-csl/yices2

    Ubuntu or Debian

    To install Yices on Ubuntu or Debian, do the following:

     sudo apt install software-properties-common
 sudo add-apt-repository ppa:sri-csl/formal-methods
 sudo apt-get update
 sudo apt-get install yices2-dev

  2. Install the following python packages:

     pip3 install termcolor yices z3-solver antlr4-python3-runtime sympy numpy bokeh scipy
  3. Download the source code (stlmc.zip) and unzip it.

  4. Run the following commands:

     cd stlmc
 make
  5. Restart the terminal.
  6. Finish!!
